# Enterprise Todo App – Backend + Frontend

Ett enkelt enterprise-projekt byggt med Spring Boot, PostgreSQL, JWT-auth och en lätt HTML/JS frontend.

## Funktioner

Registrering och inloggning med JWT

Skapa, visa, uppdatera och ta bort todos

Roller och grundläggande säkerhet

Docker-stöd för backend + databas

Enkel frontend (HTML, CSS, JavaScript)

## Tekniker

Java 17 / Spring Boot 3

Spring Security + JWT

PostgreSQL

Docker & Docker Compose

HTML / CSS / Vanilla JavaScript
